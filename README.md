# discord-bot

A Discord bot for managing tickets, deposits, withdrawals, and gambling features.

## Features
- Ticket system for deposits and withdrawals
- Coinflip gambling command
- Balance tracking with gamble requirements
- Owner commands for managing user balances
- Error handling and input validation

## Setup
1. Install dependencies: `pip install discord.py`
2. Update the bot token in `bot.py`
3. Configure the IDs in the CONFIG section
4. Run: `python bot.py`

## Commands
See `!assist` command in the bot for full list of commands.

## Code
The bot code is available in `bot.py`.
